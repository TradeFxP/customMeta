## 🔧 Quick Customization

### Change Company Name
Edit `Footer.jsx`:
```javascript
<h2 className="logo-text">YOUR_COMPANY</h2>
```

### Update Services
Edit `Services.jsx` → `servicesData` array:
```javascript
const servicesData = [
  { title: 'Your Service 1', ... },
  { title: 'Your Service 2', ... }
];
```

### Change Team Members
Edit `Team.jsx` → `teamMembers` array:
```javascript
const teamMembers = [
  { name: 'Your Name', role: 'Your Role', ... }
];
```

### Update Colors
Search and replace in all CSS files:
- `#667eea` → Your primary color
- `#764ba2` → Your secondary color
